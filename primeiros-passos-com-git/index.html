Comando básicos do git

formas de adicionar um arquivo ao stage

  • git add .

    • Adciona todos os arquivos

  • git add nome-do-arquivo

    • Adciona apenas o arquivo especificado

  • git commit -m "mensagem do commit"

    • a flag -m é usada para deixar uma mensagem

comandos para obter alguns status

  • git status

    • Como o nome sugere, ele exibe o status dos arquivos

  • git log - Exibe informações sobre os commits realizados como:
    • hash do commit
    • autor
    • data
    • Nome do commit

Configurações do git

Configurações utilizadas quando se está configurando o ambiente inicialmente

  • Configurar user name do usuario

    • git config --global user.name "seu nome de usuario do github"

    • git config --global user.email "seu email do github"
